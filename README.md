# 🌐 Link Sharing App - Frontend

This is the frontend of the **Link Sharing App**, built using **Next.js** and **Tailwind CSS**, offering a responsive UI for users to manage and share their social/content links.

It features seamless Google auth, OTP login, link editing, and drag-and-drop reordering — all with pixel-perfect responsiveness based on Figma design.

---

## ✨ Features

- Google Sign-In
- Email & Password Sign-Up / Login
- OTP-based login & verification
- Link management:
  - Add, Edit, Delete, Reorder
- Drag-and-drop functionality
- Mobile preview mockup
- Fully responsive layout
- Validations with user feedback
- Clean and accessible UI

---

## 🛠️ Tech Stack

- Next.js (React)
- Tailwind CSS
- TypeScript
- React Hook Form + Zod (for validation)
- Axios (for API requests)

---

/public
